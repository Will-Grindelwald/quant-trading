## 🛠️ 使用方式

### 快速体验
```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 运行演示（使用模拟数据）
python examples/quick_start.py

# 3. 运行测试
pytest tests/ -v
```

### 真实回测
```bash
# 1. 下载历史数据
python examples/download_data.py

# 2. 查看数据情况
python examples/check_data.py

# 3. 运行回测
python examples/run_backtest.py
```
